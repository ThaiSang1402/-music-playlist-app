# 🚀 Quick Start - Kết nối Genymotion

## Bước 1: Chuẩn bị
- ✅ Mở Genymotion device "My Phone"
- ✅ Cài **Expo Go** từ Play Store trên device

## Bước 2: Chạy ứng dụng

**Double-click file:** `expo-start.bat`

Hoặc trong terminal:
```bash
.\expo-start.bat
```

## Bước 3: Kết nối

1. Mở **Expo Go** app trên Genymotion
2. Quét **QR code** từ terminal
3. Xong! 🎉

---

## Nếu QR không hoạt động:

```bash
npm run start:tunnel
```
